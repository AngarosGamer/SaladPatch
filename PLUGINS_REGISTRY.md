## Getting Started

### To Use a Plugin/Theme

1. For official plugins/themes: Copy the file from `available-plugins/` or `available-themes/` into the `plugins/` or `themes/` folder
2. For community plugins/themes: Follow the instructions provided in the repository or documentation
3. Restart SaladPatch to load the plugin/theme

### To Create a Plugin/Theme

Start with one of these templates:
- **Plugin**: [available-plugins/plugin-template-core-log.js](available-plugins/plugin-template-core-log.js)
- **Theme**: [available-themes/catppuccin-frappe.js](available-themes/catppuccin-frappe.js)

Then read [CONTRIBUTING.md](CONTRIBUTING.md) and [PLUGIN_CHECKLIST.md](PLUGIN_CHECKLIST.md).
