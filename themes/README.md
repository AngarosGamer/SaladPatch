# SaladPatch Themes

This folder is for active themes that are loaded by the injector.

If you want to browse examples or alternates, see [available-themes](../available-themes).

## Available Themes

- [catppuccin-frappe](../available-themes/catppuccin-frappe.js): Catppuccin Frappe theme variant.
- [catppuccin-latte](../available-themes/catppuccin-latte.js): Catppuccin Latte theme variant.
- [catppuccin-macchiato](../available-themes/catppuccin-macchiato.js): Catppuccin Macchiato theme variant.
- [catppuccin-mocha](../available-themes/catppuccin-mocha.js): Catppuccin Mocha theme variant.
